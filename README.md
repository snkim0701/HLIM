# HLIM
## HBM Leading Indicator Model

**Event-driven AI Semiconductor Prediction Framework**

---

## Project Overview

HLIM (HBM Leading Indicator Model) is a research project for predicting the future state of the AI semiconductor industry using earnings events, management guidance, and market reactions.

Instead of predicting stock prices directly, HLIM estimates the market state through event analysis.

The framework is designed to support research on:

- Micron Technology
- SK hynix
- Samsung Electronics
- NVIDIA
- TSMC
- AI semiconductor ecosystem

---

## Project Goals

The project consists of five development stages.

### Version 1.0
Event Knowledge Base (EKB)

Build a structured database containing

- Earnings Events
- Guidance
- Market Reactions
- Event Returns

---

### Version 2.0
Feature Engineering

Develop quantitative features such as

- Revenue Surprise
- EPS Surprise
- Guidance Reliability Index (GRI)
- Rolling Bias
- HBM Demand Score
- Management Confidence Score

---

### Version 3.0
Verification

Evaluate predictive power using

- Event Study
- Walk-forward Validation
- Information Coefficient (IC)
- Quantile Analysis

---

### Version 4.0
Prediction Engine

Develop AI models including

- Random Forest
- XGBoost
- LightGBM
- Transformer

---

### Version 5.0
HLIM Framework

Integrate

- Event Analysis
- GRO (Global Rotation Oscillator)
- Foreign Investor Flow
- Sector Rotation

to estimate the AI Semiconductor Market State.
